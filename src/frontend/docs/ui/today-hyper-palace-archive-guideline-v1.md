# `/today` 한국적 고도화 가이드 v1

부제: **Time Descent: Hyper Palace Archive**

---

## 0. 최상위 선언

이 서브 프로젝트는 `전통을 보여주는 화면`이 아니라,  
`역사를 공부하는 감각 자체가 한국적으로 느껴지는 화면`이어야 한다.

핵심은 문양 추가가 아니라 다음 3가지를 UI 문법으로 번역하는 것이다.

- 궁중의 질서감
- 박물관의 기록성
- 라이프스타일 브랜드의 재질감

---

## 1. 분석 요약

- 한국적 고급감은 장식보다 `격식 있는 프레이밍`과 `정확한 분류`에서 나온다.
- 핵심은 많은 정보를 한 번에 보여주는 것이 아니라 `큐레이션된 경로`를 제시하는 것이다.
- 전통을 직접 복제하지 않고 현대적으로 압축해야 촌스러움을 피할 수 있다.
- 페이지는 기능 섹션이 아니라 `순차 경험 공간`처럼 읽혀야 한다.

## 2. 핵심 인사이트

## 2.1 한국의 멋은 장식보다 격식

우선 적용 요소:

- 격식 있는 제목 체계
- 전시처럼 큐레이션된 흐름
- 기록물 같은 정보 구획
- 공간처럼 느껴지는 레이어 전환

## 2.2 최적 조합

- 고궁박물관: 왕실적 중심축, 정숙한 위계, 제목 품격
- 중앙박물관: 하이라이트 중심 큐레이션
- 설화수: 전통의 미래형 번역
- 오설록: 섹션을 `공간`으로 인식시키는 네이밍/동선

## 2.3 `/today`의 경험 정의

`문제 개수 확인` 페이지가 아니라 아래 의식(ritual) 루프로 설계한다.

1. 오늘의 시대에 들어간다
2. 시간축에서 위치를 확인한다
3. 끊긴 흐름을 복원한다
4. 기록을 챙긴다
5. 다음 시대로 넘어간다

---

## 3. 결론: 고정 방향

`/today`는 아래 구조로 잠근다.

- 위: 왕실적 정렬감
- 중간: 전시형 시간축
- 아래: 아카이브형 기록층
- 끝: 현대적 학습 CTA 복귀

정식 표현:

**Today = 궁중의 질서로 시작하는 학습 현판 + 박물관식 시간축 + 기록 보관층 + 현대적 CTA 복귀**

## 4. 실행 액션 잠금

## 4.1 섹션 네이밍 재정의

기능명보다 공간명/층위명으로 고정한다.

- Hero -> `오늘의 장`
- Time Rail -> `시대의 자리`
- Flow -> `복원 순서`
- Chain -> `잇는 고리`
- Archive -> `기억 보관층`

## 4.2 Hero 구조 잠금 (왕실 현판 원리)

- 큰 제목: `오늘의 장`
- 메인 앵커: `일제강점기`
- 서브 앵커: `침탈·저항`
- 하단 메타: `1910-1945 / 연결 3 / 복습 12 / 새 문제 5`

원칙: 카드 다발이 아니라 `한 시대가 오늘의 무대`로 서야 한다.

## 4.3 시간축 잠금 (전시 도록 원리)

- 전체 레일은 얇고 길게
- 오늘 하이라이트 시대 1개만 강하게
- 연결 시대 2개만 중강도 프리뷰
- 나머지 시대는 저강도 노드

원칙: 전체를 다 보여주되 산만하게 만들지 않는다.

## 4.4 Chain 카드 잠금 (전시 캡션 패널 원리)

- 좌상단 분류표: `시대 / 축`
- 중앙 대제목: 연결 주제
- 본문: 인과 사슬 (`A -> B -> C`)
- 하단: `시험에서 같이 볼 축`
- 최하단: 왜 지금 보면 좋은지 한 줄 주석

원칙: SaaS 카드가 아니라 `작은 전시 설명판`처럼 보이게 한다.

## 4.5 기록층 잠금 (사료 보관함 원리)

`대표 유물 / 시험 포인트 / 자주 같이 보는 흐름 / 다음 연결`을 분산 카드로 두지 않는다.

- 하나의 보관 패널 안에서 인덱스 구조로 정리
- 끝 CTA만 현대적 행동 버튼으로 복귀

## 4.6 색 잠금

- 배경: 왕실 먹색 계열 진회흑
- 본문면: 백자 감각 회백
- 구획선: 묵색/청회색
- 강조: 황동/탁금 소량
- 보조 포인트: 주홍 소량

금지:

- 전면 베이지
- 과한 단청 원색
- 붓글씨 무드

## 4.7 타이포 잠금 (현판 리듬)

- 폰트 장식보다 자간/여백/정렬로 품격 형성
- 시대명은 수치/버튼보다 우선순위가 높아야 함
- 메타 정보는 표찰처럼 얇고 정돈되게 처리

## 4.8 모션 잠금

- Hero: 빠르고 현대적
- Time Rail: 부드러운 연결
- Chain: 연결선 지연 노출
- 기록층: 거의 정적

원칙: 아래로 갈수록 조용해져야 한다.

---

## 5. 리스크와 금지

## 리스크

1. 박물관처럼 과도하게 무거워져 학습 행동성이 약해질 위험
2. 럭셔리 무드 과잉으로 교육 플랫폼 정체성 약화 위험
3. 공간명만 바꾸고 실제 구조를 안 바꾸는 이름놀이 위험

## 금지

- 무드 차용만 하고 구조/위계를 고치지 않는 방식
- 전통 장식을 직접 노출하는 즉물적 연출
- 설명 문장으로만 한국성을 전달하려는 접근

---

## 6. 최종 잠금 문장

`/today`는 한국풍 장식 화면이 아니라  
**한국적 학습 경험으로 설계된 시간 하강형 학습 데스크**여야 한다.

즉,

- 궁중의 질서로 시작하고
- 박물관의 기록층으로 내려가며
- 마지막엔 현대적 학습 행동으로 복귀한다.
